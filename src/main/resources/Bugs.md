### 错误列表
- Basic
	1. 多重声明 
	2. 没有声明
	3. 使用时没有初始化
	3. 类型转换（先进行自动转换）
	4. if 和 while中的条件语句 出现类型问题，不为int
	5. 除零错误
	6. 数组索引越界
	7. 数组下标不是整数，不是正数
	8. 简单变量初始化为数组变量
	9. 数组变量和简单变量名相同
- Extend
	1. 死循环 （条件恒为 1 或 0）
	2. 常量左值不可修改
	3. 函数未声明，参数列表不匹配
